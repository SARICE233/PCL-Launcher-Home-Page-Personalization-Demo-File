# WTU-MC PCL 自定义主页

武汉纺织大学 Minecraft 社区 · PCL2 启动器自定义主页。

## 使用方法

### 联网自动更新（推荐）

1. 打开 PCL → 设置 → 个性化 → 主页
2. 选择 **"联网更新"**
3. 填入以下地址：

```
https://raw.githubusercontent.com/SARICE233/PCL-Launcher-Home-Page-Personalization-Demo-File/main/Custom.xaml
```

之后每次启动 PCL 都会自动检查更新，无需手动替换文件。

### 离线使用

下载 `Custom.xaml`，拖拽到 PCL 窗口即可。
